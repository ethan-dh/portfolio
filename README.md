## ⚙️ Installation locale

Pour installer et exécuter ce projet localement, suivez ces étapes :

1. Installer [Node.js](https://nodejs.org/fr)

2. Faire `npm install`

3. Lancer le site: `npm start`

4. Compiler le site: `npm run build`
